# ReidTracker
Tracker with reidentification


# Usage  


1) Get model:
$ sh ./models/get_models.sh

2) Install:
2.1) Use docker, really (or get dependencies from Dockerfile)
2.2) If you using it in google colab, then run:

$ pip install -r requirements.txt

$ git clone https://github.com/KaiyangZhou/deep-person-reid.git && \
    cd deep-person-reid && \
    python3 setup.py develop
